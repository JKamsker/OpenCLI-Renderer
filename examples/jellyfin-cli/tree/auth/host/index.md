# `auth host`

- Root: [index](../../index.md)
- Parent: [auth](../index.md)

Manage configured hosts

## Subcommands

- [alias](alias/index.md) — Manage host aliases
- [delete](delete.md) — Remove a host and all its profiles
- [list](list.md) — List all configured hosts
- [rename](rename.md) — Rename a host key
- [use](use.md) — Set the default host

## Examples

- `auth host list --server http://localhost:8096`
