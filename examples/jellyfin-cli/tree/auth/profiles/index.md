# `auth profiles`

- Root: [index](../../index.md)
- Parent: [auth](../index.md)

Manage saved server profiles

## Subcommands

- [delete](delete.md) — Delete a profile
- [list](list.md) — List all hosts and profiles
- [rename](rename.md) — Rename a profile
- [show](show.md) — Show the resolved profile for the current context
- [use](use.md) — Set the default profile for a host

## Examples

- `auth profiles list --server http://localhost:8096`
