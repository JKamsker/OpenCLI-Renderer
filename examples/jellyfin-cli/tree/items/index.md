# `items`

- Root: [index](../index.md)
- Parent: [](../index.md)

Browse, search, refresh, and update library items

## Subcommands

- [ancestors](ancestors.md) — Show parent hierarchy for an item
- [counts](counts.md) — Show library item counts
- [dates](dates.md) — Show date fields used for latest-items diagnostics
- [delete](delete.md) — Delete an item from the library and filesystem
- [download](download.md) — Download original media or file
- [explain-latest](explain-latest.md) — Explain why an item is or is not visible in latest
- [favorite](favorite.md) — Mark or unmark favorites
- [get](get.md) — Show one item by id
- [images](images/index.md) — Manage item artwork
- [latest](latest.md) — Inspect Jellyfin's latest-items shelf
- [list](list.md) — Query items with filters
- [lyrics](lyrics/index.md) — Read, search, download, upload, or delete lyrics
- [nfo](nfo/index.md) — Inspect local NFO sidecar metadata
- [playback-info](playback-info.md) — Inspect playback media sources and streams
- [refresh](refresh.md) — Refresh metadata, images, or trickplay
- [remote-images](remote-images/index.md) — Browse or download remote item artwork
- [remote-search](remote-search.md) — Search external metadata providers (TMDb, AniDB, IMDB, ...)
- [resume](resume.md) — Show continue-watching items
- [search](search.md) — Search names and titles
- [similar](similar.md) — Find similar items
- [subtitles](subtitles/index.md) — Search, download, upload, or delete subtitles
- [suggestions](suggestions.md) — Show suggested items for the current user
- [themes](themes.md) — List theme songs and videos for an item
- [tree](tree.md) — Show a recursive item tree for diagnostics
- [update](update.md) — Update item metadata

## Examples

- `items list --parent 12345`
