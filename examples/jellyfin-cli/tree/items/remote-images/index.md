# `items remote-images`

- Root: [index](../../index.md)
- Parent: [items](../index.md)

Browse or download remote item artwork

## Subcommands

- [download](download.md) — Download a remote image
- [list](list.md) — List available remote images
- [providers](providers.md) — List remote image providers

## Examples

- `items remote-images list 12345`
