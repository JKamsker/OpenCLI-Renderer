# `items subtitles`

- Root: [index](../../index.md)
- Parent: [items](../index.md)

Search, download, upload, or delete subtitles

## Subcommands

- [delete](delete.md) — Delete an external subtitle file
- [download](download.md) — Download a remote subtitle
- [search](search.md) — Search remote subtitle providers
- [upload](upload.md) — Upload an external subtitle file

## Examples

- `items subtitles search 12345 en --perfect-match`
