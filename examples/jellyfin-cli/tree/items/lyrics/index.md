# `items lyrics`

- Root: [index](../../index.md)
- Parent: [items](../index.md)

Read, search, download, upload, or delete lyrics

## Subcommands

- [delete](delete.md) — Delete external lyrics
- [download](download.md) — Download remote lyrics
- [get](get.md) — Get lyrics for an audio item
- [search](search.md) — Search remote lyric providers
- [upload](upload.md) — Upload an external lyric file

## Examples

- `items lyrics get 12345`
