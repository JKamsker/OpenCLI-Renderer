# `library`

- Root: [index](../index.md)
- Parent: [](../index.md)

Scan libraries and manage virtual folders [[admin]]

## Subcommands

- [folders](folders/index.md) — List, create, rename, or remove virtual folders [[admin]]
- [media](media.md) — Show media folders
- [options](options.md) — Show or update library options for one virtual folder
- [paths](paths/index.md) — Manage media paths inside a virtual folder [[admin]]
- [scan](scan.md) — Start a full library refresh

## Examples

- `library scan --server http://localhost:8096`
