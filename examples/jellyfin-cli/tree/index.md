# jf

- Version: `0.1.0`
- OpenCLI: `0.1-draft`

Manage your Jellyfin server from the command line. Available command areas include authentication, items, library management, server administration, users, playlists, and more.

<a id="overview"></a>
## Overview

### CLI Scope

- Top-level command groups: `20`
- Documented commands: `193`
- Leaf commands: `153`

### Available Commands

- [artists](artists/index.md) — Browse music artists or album artists
- [auth](auth/index.md) — Sign in, inspect identity, and manage API keys
- [backups](backups/index.md) — Create and restore backups [[admin]]
- [collections](collections/index.md) — Create collections and add or remove items
- [devices](devices/index.md) — Inspect or remove registered devices [[admin]]
- [genres](genres/index.md) — Browse library genres
- [items](items/index.md) — Browse, search, refresh, and update library items
- [library](library/index.md) — Scan libraries and manage virtual folders [[admin]]
- [livetv](livetv/index.md) — Channels, guide, recordings, timers, tuners, listings
- [packages](packages/index.md) — Browse repositories and install packages [[admin]]
- [persons](persons/index.md) — Browse actors, directors, and other people
- [playlists](playlists/index.md) — Create playlists and manage items or shares
- [plugins](plugins/index.md) — Configure, enable, disable, or uninstall plugins [[admin]]
- [raw](raw/index.md) — Low-level endpoint access
- [server](server/index.md) — Health, logs, config, restart, and shutdown
- [sessions](sessions/index.md) — Inspect sessions and remote-control clients
- [studios](studios/index.md) — Browse library studios
- [syncplay](syncplay/index.md) — Manage SyncPlay groups
- [tasks](tasks/index.md) — View and run scheduled tasks [[admin]]
- [users](users/index.md) — Manage Jellyfin users [[admin]]
