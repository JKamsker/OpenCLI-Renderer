# `sessions`

- Root: [index](../index.md)
- Parent: [](../index.md)

Inspect sessions and remote-control clients

## Subcommands

- [command](command.md) — Send a general command payload
- [list](list.md) — List active sessions
- [message](message.md) — Display a message on a client
- [play](play.md) — Start playback on a session
- [state](state.md) — Send pause/resume/stop/seek commands

## Examples

- `sessions list --controllable-by 00000000-0000-0000-0000-000000000001`
