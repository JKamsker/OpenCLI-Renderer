# `packages`

- Root: [index](../index.md)
- Parent: [](../index.md)

Browse repositories and install packages [[admin]]

## Subcommands

- [cancel](cancel.md) — Cancel an in-progress package installation
- [get](get.md) — Show details for one package
- [install](install.md) — Install a package
- [list](list.md) — List packages available from configured repositories
- [repos](repos/index.md) — Inspect or replace configured package repositories [[admin]]

## Examples

- `packages list --server http://localhost:8096`
