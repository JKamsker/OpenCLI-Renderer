# `server localization`

- Root: [index](../../index.md)
- Parent: [server](../index.md)

List reference localization data

## Subcommands

- [countries](countries.md) — List known countries
- [cultures](cultures.md) — List supported cultures
- [ratings](ratings.md) — List known parental ratings

## Examples

- `server localization cultures --server http://localhost:8096`
