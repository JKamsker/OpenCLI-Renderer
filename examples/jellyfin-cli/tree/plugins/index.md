# `plugins`

- Root: [index](../index.md)
- Parent: [](../index.md)

Configure, enable, disable, or uninstall plugins [[admin]]

## Subcommands

- [config](config/index.md) — Read or update plugin configuration [[admin]]
- [disable](disable.md) — Disable a plugin
- [enable](enable.md) — Enable a plugin
- [list](list.md) — List installed plugins
- [uninstall](uninstall.md) — Uninstall a plugin

## Examples

- `plugins list --server http://localhost:8096`
