# `playlists items`

- Root: [index](../../index.md)
- Parent: [playlists](../index.md)

Manage playlist items

## Subcommands

- [add](add.md) — Add items to a playlist
- [list](list.md) — List playlist items
- [move](move.md) — Move an item within a playlist
- [remove](remove.md) — Remove items from a playlist

## Examples

- `playlists items list 12345`
