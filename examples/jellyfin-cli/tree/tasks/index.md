# `tasks`

- Root: [index](../index.md)
- Parent: [](../index.md)

View and run scheduled tasks [[admin]]

## Subcommands

- [get](get.md) — Show one task
- [list](list.md) — List tasks
- [start](start.md) — Start a task
- [stop](stop.md) — Stop a running task
- [triggers](triggers.md) — Show task triggers or replace them with 'tasks triggers set <TASK_ID>'

## Examples

- `tasks list --server http://localhost:8096`
