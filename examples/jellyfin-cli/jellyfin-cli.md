- [backups](#command-backups) — Create and restore backups [[admin]]

- [items](#command-items) — Browse, search, refresh, and update library items
- [library](#command-library) — Scan libraries and manage virtual folders [[admin]]
- [livetv](#command-livetv) — Channels, guide, recordings, timers, tuners, listings
- [packages](#command-packages) — Browse repositories and install packages [[admin]]
- [persons](#command-persons) — Browse actors, directors, and other people
- [playlists](#command-playlists) — Create playlists and manage items or shares
- [plugins](#command-plugins) — Configure, enable, disable, or uninstall plugins [[admin]]
- [raw](#command-raw) — Low-level endpoint access
- [server](#command-server) — Health, logs, config, restart, and shutdown
- [sessions](#command-sessions) — Inspect sessions and remote-control clients
- [studios](#command-studios) — Browse library studios
- [syncplay](#command-syncplay) — Manage SyncPlay groups
- [tasks](#command-tasks) — View and run scheduled tasks [[admin]]
- [users](#command-users) — Manage Jellyfin users [[admin]]

<a id="command-auth"></a>
## `auth`

Sign in, inspect identity, and manage API keys

### Subcommands

- `host` — Manage configured hosts
- `keys` — Manage API keys [[admin]]
- `login` — Sign in with username and password
- `logout` — Remove stored credentials and end the current session
- `profiles` — Manage saved server profiles
- `quick` — Quick Connect flows
- `users` — List public users shown on the login screen
- `whoami` — Show the authenticated user and current server

### Examples

- `auth login --username demo --password secret`
