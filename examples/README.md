# Example Renders

This folder contains tracked Markdown example renders generated by `opencli-renderer` from sibling CLIs in the same workspace.

- `jellyfin-cli/` — output generated from `..\Jellyfin-Cli`
- `jdownloader-remotecli/` — output generated from `..\JDownloader-RemoteCli`

Each example includes:

- `opencli.json`
- `xmldoc.xml`
- a single-file Markdown render
- a tree Markdown render
