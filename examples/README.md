# Example Renders

This folder keeps checked-in source snapshots and Markdown renders generated by `inspectra-gen`.

- `jellyfin-cli/` comes from `..\Jellyfin-Cli`
- `jdownloader-remotecli/` comes from `..\JDownloader-RemoteCli`
- `inspectra-gen/` documents this repository's own CLI surface

Each example directory may include:

- `opencli.json`
- `xmldoc.xml`
- a single-file Markdown render
- a tree Markdown render

HTML output is no longer checked in here as standalone files. The Pages workflow publishes HTML examples as bundle directories with `index.html` plus versioned assets.
